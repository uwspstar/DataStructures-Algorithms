# Knuth-Morris-Pratt 字符串查找算法 简称为 KMP 算法
- 可在一个字符串S内查找一个词W的出现位置。一个词在不匹配时本身就包含足够的信息来确定下一个匹配可能的开始位置，此算法利用这一特性以避免重新检查先前匹配的字符。
