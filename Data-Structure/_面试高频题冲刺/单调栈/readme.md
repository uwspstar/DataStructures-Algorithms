# O(n) 的算法 : n 次并查集，n 次哈希表，单调栈

# 什么是单调栈
- 单调栈算法是一种借助“栈” 实现的算法 特征为栈内元素按照某种规则(一般为数字大小)单调 如:“[1, 3, 7” 或 “[6, 2, 0”

# 单调栈的最常见用法
- 对于数组中的某个元素，在数组中找到它左侧(或右侧)离 它最近的一个比它大(或比它小)的数字
